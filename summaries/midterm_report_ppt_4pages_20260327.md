# 中期报告（论文/答辩PPT四页结构）

## 第1页：摘要（Abstract）

题目：面向 VisDrone 密集小目标场景的零样本检测增强研究（中期）

研究目标：
在不大规模重训主干模型的约束下，提升无人机场景中密集小目标检测效果，建立可复现、可扩展的实验闭环。

当前进展：
1. 已完成从数据子集构建、批量推理、统一评估到结果归档的全流程。
2. 已系统完成整图基线、阈值搜索、滑窗网格、融合策略、外接模块训练与主模型小样本微调。
3. 形成多组同口径对照，指标可复现，工程链路稳定。

阶段性结论：
1. 最优综合结果达到 Acc@0.5=0.4113，mAP@0.5=0.1685。
2. 相比默认基线，Acc@0.5 提升 0.1036，mAP@0.5 提升 0.0318。
3. 当前瓶颈已从流程搭建转向候选框覆盖与排序优化的上限约束。

---

## 第2页：方法（Method）

总体思路：
以 GroundingDINO 为主干，采用“推理策略优化 + 轻量训练增强”双线推进。

方法流程：
1. 基线建立：默认整图推理，确定统一评估口径（class-aware，Acc@0.5/0.75，mAP@0.5）。
2. 参数优化：进行 box/text/nms 阈值搜索，得到 best 整图配置。
3. 滑窗增强：构建 3×3 滑窗网格（窗口尺寸×重叠率），分析 Acc 与 mAP 的最优配置分化。
4. 融合策略：将整图 best 与滑窗 best 候选框融合，形成 Stage A 保底增强。
5. 外接训练：训练轻量校准头（SL+RL），在不改主干或少改主干条件下优化候选框保留与排序。
6. 微调对照：进行小样本主模型微调（head_plus），再接入外接模块，验证全链路增益。

技术特点：
1. 保留主干稳定性，优先采用低成本可插拔改进。
2. 全流程指标统一，便于横向对比与复现。
3. 同时关注检测精度与推理开销，兼顾可部署性。

4. Qwen 系列在当前检测协议下显著弱于 GroundingDINO 主线，验证了路线选择合理性。

## 第4页：结论与后续计划（Conclusion）

中期结论：
1. 项目已完成从基线到增强模型的可复现实验闭环。
2. 当前阶段实现了稳定、可解释的精度提升，核心指标达到阶段预期。
3. 当前性能上限受候选框覆盖约束明显，后续需从“候选生成质量”而非仅“候选重排”突破。

主要贡献：
1. 构建了统一评估口径与多阶段可追溯实验体系。
2. 证明了轻量外接训练在不大改主干时的有效性。
3. 通过微调主模型+外接模块实现当前最佳综合结果。

下一阶段计划：
1. 以 Exp-14 为主线进行多随机种子复现，验证稳定性与显著性。
2. 做双目标搜索（Acc@0.5 与 mAP@0.5 联合）以缓解单指标牺牲。
3. 引入更强候选框生成策略，突破 Acc 平台期。
4. 将精度-速度-显存纳入统一评分，形成部署导向方案选择标准。
